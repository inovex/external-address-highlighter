# External Address Highlighter

This is a web-extension.

# Prerequisites

- node v8.9.1
- npm v5.5.1


# Commands

`$ npm start`

`$ npm run serve`

`$ npm run lint`

`$ npm run build`

`$ npm run sign -- --api-key=user:00000000:000 --api-secret=asdasdasdasdasdasdasdasdasdasdasd`


# Documentation

see https://developer.mozilla.org/en-US/Add-ons/WebExtensions
