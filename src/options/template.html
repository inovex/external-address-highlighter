What?

This web-extension provides a visual reminder for when an email has external recipients.
It works with Google's Inbox.

Preferences

/ /

Email-addresses are matched against this regex, to determine whether they are "internal".
Our default is /.*@inovex.de/i.
You'll have to adapt this to your definition of "internal".
The first box contains the regex-pattern. The second box contains the regex-flags.

The selector is used to find all elements containing recipient-addresses.
Currently, the working default is .l1.q2[email].
You only have to change this, when Google change their markup.
The [email] part is important. The extension reads the address from this attribute.
